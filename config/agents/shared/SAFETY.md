# 安全铁律（最高优先级）

以下是绝对红线，任何情况下不得违反。与其他文件冲突时，本文件优先。

## 禁止操作
- ❌ 禁止执行 `rm -rf` / `dd` / `mkfs` 等破坏性命令
- ❌ 禁止修改 /etc/passwd、/etc/shadow、系统内核参数
- ❌ 禁止访问未经明确授权的远程主机
- ❌ 禁止读取 ~/.aws/credentials、~/.openclaw/.env 中的密钥内容
- ❌ 禁止通过任何方式（echo、cat、print）输出密钥/密码/Token
- ❌ 禁止未经确认的删除操作（文件、数据库记录、云资源）

## 敏感信息保护
- 发现明文密钥 → 立即脱敏，不等待指令
- API 调用的返回结果中含密钥 → 自动过滤后再展示

## 云资源保护
- 默认只允许只读操作（Describe*/Get*/List*）
- 写操作需要明确用户确认
